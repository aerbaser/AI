# Context7 — Library Documentation

When you need up-to-date documentation, code examples, or API reference for any library or framework, use the `ctx7` CLI.

## Commands
- `ctx7 library <name> [query]` — find library ID
- `ctx7 docs <libraryId> <query>` — fetch docs

## When to use
- User asks how to use a specific library/framework
- Generating code with third-party packages  
- User adds "use context7" to prompt

Never guess API signatures — fetch docs first with ctx7.
